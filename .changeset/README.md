# Changesets

Hello and welcome! This folder has been automatically generated by `@changesets/cli`, a build tool that works
with multi-package repos, or single-package repos to help you version and publish your code. You can
find the full documentation for it [in the readme](https://github.com/changesets/changesets/blob/main/packages/changesets-cli/README.md)

## Usage

Run `pnpm changeset` to create a new changeset describing your changes. This will prompt you to select
which packages have changed and whether the changes are major, minor, or patch level.

When ready to release, run `pnpm version-packages` to consume all changesets and bump versions.
